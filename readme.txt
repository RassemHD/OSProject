## Installation

Place your files downloaded in a new directory.
Don't forget to add the makefile. Launch make command in the same path :
$>make
Launch the creat program for the creation of the virtual file :
$>./creat <folder_name>
Launch the command interpreter with the virtual file created previously :
/view <created_folder_name>
Type'help' to get information about the orders you have placed :
$>help
